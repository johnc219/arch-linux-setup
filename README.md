## dotfiles

These are my Arch Linux public dotfiles:
- bashrc
- fehbg
- xinitrc
- Xmodmap
- Xresources
- xserverrc
- vimrc

## packages
- xorg
- i3
- urxvt
- feh
- redshift

## usage
1. clone this repo
1. run `.make.sh` to set up dotfiles. Dotfile symlinks will be created in `~/dotfiles`. Old dotfiles will be placed in `~/dotfiles_old`
